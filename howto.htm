How was this site made?

This website was created using HTML and CSS. HTML is a website format that can be edited using a plain text editor such as Notepad.
CSS stands for Cascading Style Sheets and can be used to structure and format HTML web pages.
HTML can be used to bold text, italicize text, and more.

The green background color you see right now is possible because of CSS! Otherwise, you'd be looking at a boring, white background instead.
